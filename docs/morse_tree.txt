MORSE CODE TREE  —  cabinet reference
======================================
  ATTACK = dit  (·)    JUMP = dash  (—)
  Left branch always = dit   Right branch always = dash

  RIGHT  — accept current letter, advance to next position
  LEFT   — accept and go back  (at start: switch to alpha mode)
  UP hold 0.7s — submit entire text

  Gaps (U right, R right, O left/right) have no standard letter.

Example — to type  A  (·—):
  press ATTACK (·) → lands on E
  press JUMP   (—) → lands on A
  press RIGHT  to accept A and move to next position
